# CRM - Gestión de Clientes y Notas

Una aplicación web sencilla para la gestión de clientes y sus notas asociadas, desarrollada con tecnologías web estándar.

## 🚀 Demo

🔗 **[Ver aplicación en vivo](https://app-45sp1-mryq29c89-luisernestosalas-2775s-projects.vercel.app)**

## 📋 Características

- ✅ Gestión completa de clientes (crear, editar, eliminar)
- 📝 Sistema de notas asociadas a cada cliente
- 🔍 Búsqueda y filtrado de clientes
- 💾 Almacenamiento local en el navegador
- 📱 Interfaz responsive y fácil de usar
- ⚡ Aplicación ligera sin dependencias externas

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Almacenamiento**: LocalStorage
- **Despliegue**: Vercel
