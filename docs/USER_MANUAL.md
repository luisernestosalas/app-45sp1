# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Es un sistema simple para gestionar clientes y sus notas. Puedes agregar información de clientes, crear notas para cada uno y mantener todo organizado en un solo lugar.

## Acceso a la aplicación

**URL:** https://app-45sp1-mryq29c89-luisernestosalas-2775s-projects.vercel.app

La aplicación funciona en cualquier navegador web moderno (Chrome, Firefox, Safari, Edge).

## Funciones principales

### 1. Gestión de Clientes

**Agregar un cliente nuevo:**
1. Busca el botón "Agregar Cliente" o el símbolo "+"
2. Completa los campos requeridos:
   - Nombre del cliente
   - Email (si es obligatorio)
   - Teléfono (si es obligatorio)
   - Otros datos según el formulario
3. Haz clic en "Guardar"

**Ver lista de clientes:**
- La pantalla principal muestra todos los clientes registrados
- Puedes ver información básica de cada uno

**Editar un cliente:**
1. Haz clic sobre el cliente que quieres modificar
2. Cambia la información necesaria
3. Guarda los cambios
